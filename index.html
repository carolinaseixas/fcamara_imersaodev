MAS ATENÇÃO:
para você que quer voltar para o escritório, temos algumas regras:

Será medida a temperatura ao chegar no escritório

Todos os consultores deverão usar máscaras dentro das acomodações dos escritórios

Nas mesas existem adesivos sinalizando onde os #SangueLaranjas devem sentar, para que haja o distanciamento necessário (os escritórios só podem comportar até 40% dos consultores, conforme a legislação)

Pessoas que estiverem sentindo algum sintoma não devem ir trabalhar presencialmente, a indicação é para que permaneçam em casa
